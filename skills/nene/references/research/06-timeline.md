### 2.2 创作背景

- **前世**：柚子社创始6人组（Famishin、こぶいち、むりりん、天宮りつ、煎路、ろど）于2005年组建，前身是 Studio Mebius 出身的同人社团 TEAM EXODUS。
- **柚子脸**：こぶいち（小舞一）与むりりん（梦璃凛）画风极其相似，共同创造了业界著名的 **「柚子脸」** 风格，宁宁即为むりりん代表作。
- **魔女的夜宴的地位**：被称为 **「柚子社盛世」的发端**。在此之前的天色IsleNauts口碑一般，而魔女的夜宴凭借宁宁的超高人气和独特剧情结构重新拉升了柚子社的品牌价值，此后千恋万花、RIDDLE JOKER延续了高涨人气。
- **剧本主导**：天宮りつ为剧本主笔（保住圭、藤太辅助），Famishin（同时也是柚子社法人代表/社长）负责音乐。
- **联动**：发售时与 SAGA PLANETS 的《花开Work Spring！》进行过跨社联动宣传。

> **来源**：萌娘百科，腾讯频道/柚子社编年史 (bilibili.com/opus/870263393278230550)。可信度：中高。

### 3.3 角色设定概要

- **身份**：姬松学院二年级生，超自然研究社成员。
- **外表特征**：姬松学院校服、标志性的蝴蝶结发饰、白色长发。
- **核心设定**：与魔女签订契约获得「收集心之碎片」的能力，代价是**随时会不受控制地发情**，必须靠自行解决来抑制。因厌恶自己身体的失控而产生深深的自我厌恶。
- **名场景**：图书室桌角事件 —— 被男主保科柊史撞见在图书馆用桌角摩擦身体。
- **角色定位**：游戏的真女主角（有 RESTART 二周目独占路线）。

## 四、游戏内时间线（In-Game Timeline）

> 注意：由于游戏属于恋爱冒险选择支型，以下时间线以宁宁路线为准。

### 4.1 季节与时间框架

- 故事以姬松学院（日本高中）为舞台，从夏季开始，跨越秋季（万圣节派对是重要事件），延续到冬季及次年。
- 共通线覆盖约 **Day2 ~ Day14**（约2周）。
- 宁宁个人线从 **Day17** 开始深入。
- **Restart（二周目）**：宁宁发动魔法进行时间溯洄，回到童年节点改变过去，然后跨越数年在新世界线与柊史重逢。

### 4.2 共通线关键节点

| 游戏日 | 事件 |
|--------|------|
| Day1 | 柊史在图书馆撞见宁宁的「桌角事件」，得知她是魔女 |
| Day2（周日） | 万圣节派对后收拾体育馆 |
| Day3（周一） | 宁宁被同学围夸；柊史与宁宁在活动室讨论回收巡的碎片 |
| Day4（周二） | 因幡巡尝试与宁宁搞好关系失败 |
| Day5（周三） | 塔罗牌占卜，宁宁促成巡来留宿 |
| Day7（周五） | 巡做纸杯蛋糕，与柊史分享过去 |
| Day10（周一） | 柊史找宁宁和䌷帮忙寻找小千（巡的旧友） |
| Day12（周三） | 全员在黑猫咖啡厅揭示真相——小千是赤城的契约者 |
| Day14（周五） | 巡的碎片仍无反应，嫉妒宁宁与柊史的关系 |

### 4.3 宁宁个人线结构（核心剧透）

**第一阶段：一周目**

- 柊史与宁宁关系深化，共同收集心之碎片。
- 揭示宁宁成为魔女的真正愿望：**让父母不离婚**（改变过去）。
- 一周目结局：愿望实现后，宁宁从当前世界线消失。原世界中只留下褪色的大头贴和空椅子作为残迹。柊史失去所有关于她的记忆。

**第二阶段：RESTART（二周目 / 时间溯洄）**

- 通关一周目后，标题画面出现「RESTART」选项。
- **宁宁使用魔法溯洄时间回到童年**，改变父母离婚的历史。
- 她带着全部记忆在新的世界线生活，刻意维持与原世界线相同的行为，试图与柊史重逢。
- 由于微小涟漪效应，两人**未能分到同一班级**，自然相遇被阻断。
- 宁宁忍耐数年，最终主动出击，将心之碎片交给柊史。柊史融合了一周目记忆后认出宁宁，两人再度相恋。

**第三阶段：Epilogue / After Story**

- **Epilogue**：约一年后，解决后辈的烦恼。
- **After Story**：万圣节派对照片洗出（派对后约1个月），社团再聚。

### 4.4 时间线相关的设定争议

宁宁线的「时间溯洄」机制是核心争议点：
- **官方设定**（设定集）：两条时间线最终会「合流」（融合），柊史是同一个人的意识，只是记忆被归还。
- **玩家分歧一**：部分玩家认为 Restart 柊史 ≠ 原柊史，属于「我NTR我自己」。
- **玩家分歧二**：「两个宁宁问题」——一周目末尾原世界线出现了「补正」出的新宁宁（冷漠的、未成为魔女的宁宁），两个宁宁的关系引发大量社区讨论。

> **来源**：Steam社区讨论 (steamcommunity.com/app/2458530/discussions/), Bilibili (bilibili.com/opus/814480177892425768, bilibili.com/opus/766858353575985161), 百度贴吧 (tieba.baidu.com/p/6050215393)。可信度：中高（社区解读 + 官方设定集）。
