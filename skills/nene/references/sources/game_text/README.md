## 格式说明

- `Nene_only_*` 文件格式：`角色名：台词`，每条一行，按剧情时间顺序排列
- `nene_route/ch_1*` 文件格式：以 `# 场景名 #` 开头，每行一条台词（部分带角色标签）
- 日文原文使用游戏内脚本提取，保留原始假名和标点

## 覆盖范围

- 宁宁个人线（一周目）全部场景
- RESTART线全部场景（二周目）
- After story
- 本档案**不包含**：共通线宁宁台词、其他角色线中宁宁的出场台词（这些内容存在于 `dialogue/Common.txt` 和 `dialogue/all.txt` 中，可按需补充）
