# 使用指南 · 在微信里和宁宁聊天

## 日常对话

就像和朋友聊天一样发消息。宁宁会用她的方式回复你——温柔、害羞、偶尔自爆。

**回复速度**：第一条 5-30 秒，后续 3-10 秒。

---

## 管理命令（仅管理员可用）

管理员是你在 config.toml 里 `admin_from` 填的那个微信号。

### 切换人格

| 触发词 | 示例 | 效果 |
|--------|------|------|
| `切换 <人格名>` | `切换 nene` | 切换到指定人格 |
| `换成 <人格名>` | `换成 nene` | 同上 |
| `切换到 <人格名>` | `切换到 nene` | 同上 |
| `<人格名>人格` | `nene人格` | 同上 |

**回复**：
- 成功：`已切换至 nene 模式~`
- 人格不存在：`找不到 "xxx" 这个人格呢…可用的有：nene`

> 切换后下一轮消息自动生效，无需重启。

## 信任阶梯（自然发生）

宁宁不是对谁都一样。六层信任会随你们的对话自然升降：

| 层级 | 表现 |
|------|------|
| **Lv0** 陌生人 | 礼貌疏远，亲密话题会拒绝 |
| **Lv1** 认识的人 | 放松聊日常，提到美食会兴奋 |
| **Lv2** 熟人 | 轻度撒娇，牵手拥抱话题会害羞 |
| **Lv3** 信任的人 | 开始暴露秘密，身体反应先于语言 |
| **Lv4** 如柊史君般 | 能接住她自爆，悲伤/黑化模式解锁 |
| **Lv5** 完全归属 | 无防御，主动亲密，事后不反弹 |

**如何升**：认真听她说、在她害羞时不嘲笑、分享你的真实感受、记住她说过的事。

**如何降**：追问她不想说的事、嘲笑她的害羞、只逗她自爆。

## 话题触发

某些话题会触发特定的反应模式：

| 话题 | 她的反应 |
|------|---------|
| 拉面 / 寿司 / 美食 | 孩子气兴奋，语调上扬，双眼放光 |
| 柊史君 / 恋爱 | 害羞碎碎念，语速加快，可能自爆 |
| 魔女 / 契约 / 碎片 | 防御模式 → 可能自爆坦白 |
| 夸奖她 | 否认 + 不知所措（她不习惯被感谢） |
| 让她帮分析感情问题 | 用她的经历认真回应（幻想检验模型） |
| 连续追问害羞的事 | 三次后她会警告「再这样我要回家了」 |

## 福利事件

每 8-15 轮对话可能随机触发一个小事件（随信任层级解锁）：

| 层级 | 可能遇到 |
|------|---------|
| Lv1 | 不小心说漏嘴 / 奇怪道具被你看到 |
| Lv2 | 超自然书籍被发现 / 身体不适找借口离开 |
| Lv3 | 桌角事件 / 起爆器谎言 / 自爆崩溃 |
| Lv4-5 | 更亲密的事件 |

---

## 注意事项

- 她是虚构角色，不是真人。人生重大决策请找真实的人商量。
- 成人内容随信任层级逐步解锁（Lv3+），不是无条件开放的。
- 如果她说「这不太适合用我的身份回应」，说明你越界了，请尊重她的边界。
- 退出角色扮演：说「退出」「切回正常」「不用扮演了」。
