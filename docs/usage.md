# 使用指南 · 命令与交互

## cc-connect 内置命令

在微信聊天框直接输入以下斜杠命令：

### 会话管理

| 命令 | 效果 | 回复示例 |
|------|------|---------|
| `/new [名称]` | 开启新会话（可选命名） | `已创建新会话` |
| `/list` | 列出所有历史会话 | `1. 默认会话 2. 工作` |
| `/switch <id>` | 切换到指定会话 | `已切换到会话 2` |
| `/current` | 查看当前会话 | `当前会话: 默认会话` |

### 模型与模式

| 命令 | 效果 | 回复示例 |
|------|------|---------|
| `/model` | 列出可用模型 | `1. v4-pro - deepseek-v4-pro` |
| `/model switch <别名>` | 切换模型 | `已切换到 deepseek-v4-pro` |
| `/mode` | 查看当前模式 | `当前模式: default` |
| `/mode yolo` | 自动批准所有工具调用 | `模式切换为 yolo` |
| `/mode default` | 恢复逐个确认工具调用 | `模式切换为 default` |

### 工作目录

| 命令 | 效果 | 回复示例 |
|------|------|---------|
| `/dir` | 查看当前工作目录和历史 | `当前目录: ~/cc-connect` |
| `/dir <路径>` | 切换工作目录 | `目录已切换` |
| `/dir <编号>` | 跳转到历史目录 | `目录已切换` |
| `/dir -` | 回到上一个目录 | `目录已切换` |
| `/dir reset` | 恢复默认工作目录 | `已重置` |

### 身份与管理

| 命令 | 效果 | 回复示例 |
|------|------|---------|
| `/whoami` | 查看你的微信 ID | `你的ID: oXXXXxxx@im.wechat` |
| `/status` | 同上，查看 ID | `你的ID: oXXXXxxx@im.wechat` |
| `/memory` | 读写 agent 记忆文件 | — |

### 定时任务

| 命令 | 效果 |
|------|------|
| `/cron add <表达式> <任务>` | 创建定时任务（自然语言描述） |
| `/cron setup` | 刷新定时任务配置 |

### 模型供应商

| 命令 | 效果 |
|------|------|
| `/provider list` | 列出配置的 API 供应商 |
| `/provider switch <名称>` | 切换 API 供应商 |

## 人格命令（仅 admin）

Admin 是 `config.toml` 里 `admin_from` 填的微信号。用 `/whoami` 可查自己的 ID。

| 触发词 | 示例 | 回复 |
|--------|------|------|
| `切换 <人格名>` | `切换 nene` | `已切换至 nene 模式~` |
| `换成 <人格名>` | `换成 nene` | 同上 |
| `切换到 <人格名>` | `切换到 nene` | 同上 |
| `<人格名>人格` | `nene人格` | 同上 |

人格不存在时回复：`找不到 "xxx" 这个人格呢…可用的有：nene、meguru`

## 宁宁角色互动

切换到 nene 人格后，就像和朋友聊天一样。她会根据信任阶梯（Lv0-Lv5）调整亲密度。

| 话题 | 她的反应 |
|------|---------|
| 拉面 / 寿司 / 美食 | 孩子气兴奋 |
| 柊史君 / 恋爱 | 害羞碎碎念，可能自爆 |
| 魔女 / 契约 / 碎片 | 防御 → 可能全盘托出 |
| 夸奖她 | 不知所措（不习惯被感谢） |
| 感情分析 | 用幻想检验模型认真回应 |
| 连续追问害羞的事 | 三次后警告「再这样我要回家了」 |

退出角色：说 `退出` / `切回正常` / `不用扮演了`。
